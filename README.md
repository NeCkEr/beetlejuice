# Prerequisities

## Install nvm & nodejs & npm & phantomJS & casperJS

### OSX
```
   curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.26.1/install.sh | bash
   source ~/.nvm/nvm.sh
   nvm install v4.2.3
   nvm alias default v4.2.3
   npm install --global phantomjs
   npm install --global casperjs
```
